The easiest way to load Twitter Bootstrap and Font Awesome on your WordPress site.

== Description ==

Bootstrap and Font Awesome by HocWP Team is a plugin that allows to load Twitter Bootstrap and Font Awesome on your WordPress site.

**Current Versions**

* Bootstrap: 3.3.7
* Font Awesome: 4.7.0

**Translations**

* English
* Vietnamese

== Installation ==

Install this plugin from your WordPress site Dashboard or follow these steps below:

1. Download plugin from WordPress Plugins directory and extract it.
2. Upload the `sb-tbfa` folder to the `/wp-content/plugins/` directory.
3. Activate the Bootstrap and Font Awesome by HocWP Team plugin through the `Plugins` menu in WordPress.

1, 2, 3 you are done.
